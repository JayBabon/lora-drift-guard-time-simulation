# LoRa Drift and Guard-Time Simulation

This repository contains the source code and supporting scripts for the capstone project entitled **Simulation-Based Assessment of Timing Drift and Guard-Time Parameters in LoRa Mesh Systems**.

The project uses ns-3 to simulate a tree-based multi-hop LoRa mesh network and evaluate how timing drift and guard-time configurations affect packet delivery reliability, packet loss, guard-time violations, and delay.

## Repository Structure

- `main_source_code/` – final ns-3 simulation source code
- `phase1_simulation_model_setup/` – simulation model setup and verification scripts
- `phase2_drift_impact_evaluation/` – drift-impact evaluation scripts
- `phase3_guard_time_evaluation/` – guard-time evaluation scripts
- `phase4_acceptable_gt_determination/` – acceptable guard-time determination scripts
- `phase5_strategy_validation/` – strategy formulation and validation scripts
